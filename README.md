# 危殼：自動化合集包 (Velkora: All-in-One Automation Pack)

為《危殼：崩壞現實指南》（Velkora: Guide to the Shattered Reality）打造的專屬 Foundry VTT 自動化輔助模組。本模組旨在簡化屏障壓力計算、臨界反常判定、原初四季律動以及過載/縫合等繁瑣規則，提供 DM 與玩家一個流暢、具視覺美感且自動化的現實崩壞跑團體驗。

---

## 🌟 核心功能 (Core Features)

*   **👁️ 屏障壓力與結痂監控系統 (Veil Stress & Scab Monitor)**：
    *   自動依據參戰玩家平均等級推薦基礎屏障閾值。
    *   壓力值到達總閾值時自動判定「屏障破裂 (Veil Breach)」，增加結痂數，並扣減總閾值。
    *   壓力跨越半值時自動抽取並公佈「臨界點反常 (Critical Anomaly)」。
    *   全功能內聯一鍵撤銷（Undo）按鈕，可回退壓力值、結痂數、日誌與扣除的血量。
*   **🩸 血肉縫合自動化 (Barrier Suture)**：
    *   玩家點擊特性時自動彈窗偵測當前角色剩餘法術位/契術位（Pact Slots）。
    *   自動投擲 `[環階]d6` 絕對暗蝕傷害，優先扣除臨時生命值。
    *   生命值歸零時自動對 Actor 套用「昏迷 (Unconscious)」並增加 1 層「力竭 (Exhaustion)」，同時降低當前區域的屏障壓力。
*   **⚡ 過載施法機制 (Overload Casting)**：
    *   玩家宣告過載施法時，可點選 **法術升階 (Elevation)**、**絕對貫穿 (Penetration)** 或 **極效破壞 (Destruction)**。
    *   該法術結算後會自動產生雙倍屏障壓力。
    *   **法術升階**：自動使施法威力提升 2 環（上限 9 環）。
    *   **絕對貫穿**：自動在攻擊檢定套用優勢，或使目標豁免套用劣勢。
    *   **極效破壞**：自動將該法術的所有傷害/治療骰直接取最大值結算。
*   **🌿 原初四季律動系統 (Primal Seasons Rhythm)**：
    *   開戰時為原初施法者（德魯伊/遊俠）隨機初始化一個季節，並套用常駐「四季同諧 (Harmony)」被動效果（如畸春近戰距離增加、狂夏速度增加、死秋獲得暗蝕抗性、凜冬免疫倒地）。
    *   施放法術時自動結算「季節脈衝 (Pulse)」效果並自動往後輪轉一個季節：
        *   **畸春**：自動獲得臨時生命值。
        *   **狂夏**：自動重擲傷害/治療骰中結果為 1 的骰子。
        *   **死秋**：彈出 Dialog 讓玩家選擇要無視目標的哪種傷害抗性。
        *   **凜冬**：自動在施法者腳下生成 10 呎冰霜MeasuredTemplate模板。
*   **🌐 完整多語言與手動覆蓋 (i18n & Manual Override)**：
    *   完整支援 **正體中文 (zh-tw)**、**簡體中文 (zh-cn)** 與 **英文 (en)**。
    *   提供手動語系覆蓋設定，防止 Foundry 本地環境或玩家之間語言設定不一致導致顯示混亂。

## 🛠️ 安裝教學 (Installation)

1.  開啟 Foundry VTT，進入 **Add-on Modules** 頁面。
2.  點擊 **Install Module**。
3.  在最下方的 **Manifest URL** 欄位貼上以下連結：
    ```text
    https://raw.githubusercontent.com/RaMuLHC/velkora-all-in-one/main/module.json
    ```
4.  點擊 **Install**，並在您的遊戲世界中啟用 **Velkora : All in one automation pack**。

### 📋 必要相依模組 (Dependencies)
本模組需要以下自動化套件才能正常運作，請確保已安裝並啟用：
*   `socketlib` (核心網路通訊)
*   `midi-qol` (核心流程攔截與被動判定)
*   `libWrapper` (核心 API 封裝攔截)

## 📖 使用教學 (Usage Tutorial)

### 👑 DM 使用指南
1.  **開啟監控面板**：點擊右側邊欄設定中的 **開啟危殼監控面板**，或在聊天框輸入 `/veil` 即可開啟浮動監控 HUD。
2.  **戰鬥開始**：當拉入 Token 點擊開始戰鬥時，系統會自動跳出對話框，顯示玩家平均等級並推薦基礎閾值。確認後會自動重置結痂數與壓力值。
3.  **手動調整**：在面板中可以使用 `+5 壓`、`+1 壓` 等按鈕微調。若不小心計算錯誤，可直接在面板日誌中點擊 ↩️ **撤銷** 按鈕。
4.  **環境修正**：可以在監控面板中即時修改環境修正值，系統會動態重算總閾值。
5.  **手動控制季節**：在面板上點擊施法者名字旁的 `◀` 與 `▶` 按鈕，可以神聖遙控強行變更玩家當前的季節。

### 👤 玩家使用指南
1.  **特性發送**：遊戲開始時，DM 登入後系統會自動為人物欄派發三個專屬特性：
    *   `屏障縫合` (Barrier Suture)
    *   `主動輪轉` (Active Rotation)
    *   `過載施法` (Overload Casting)
2.  **主動輪轉**：在戰鬥中，可以在特質欄點擊使用「主動輪轉」來手動變更自己的季節被動。
3.  **施法與過載**：施法前若想進行過載施法，先點擊「過載施法」特質宣告，彈出選單選擇過載類型。接著使用法術位施放法術，Midi-QOL 會自動捕獲並套用加成（如傷害最大化或升環），並自動在結算後為該區域帶來雙倍壓力。

## ⚠️ 注意事項 (Points of Attention)

1.  **玩家語系獨立**：本模組的語系覆蓋（Language Override）為客戶端獨立設定。若玩家希望使用英文，只需在其本機的模組設定中將「手動語系覆蓋」改為 `English`，不影響其他中文玩家。
2.  **戰鬥外限制**：季節輪轉與季節脈衝僅在**戰鬥狀態下**生效。在戰鬥外施法不會產生任何季節波動或壓力（但手動調整除外）。
3.  **絕對傷害機制**：屏障縫合造成的靈魂撕裂暗蝕傷害為**絕對傷害**，會直接繞過任何抗性或免疫。
